Cryptoshop is a little cryptographic tool written in Python 3 and PyQt5 for the gui. It can encrypt, sign, generate assymetric key pairs etc... with GnuPG. GnuPG command line tool need to be installed. On Windows system, you can install Gpg4win.

ENCRYPTION

Cryptoshop can encrypt files in various strong cryptographics formats. Serpent 256, AES 256, and ThreeFish 512. Serpent and ThreeFish use mode GCM and EAX. Authentification is included in this recents modes.
AES use the mode CTR-BE. There is no authentification in this mode. Cryptoshop use an HMAC SHA-256 for authentication. This is one of the best method.

Cryptoshop is fast. It use the very good C++ crypto library Botan.

NOTE ON SECURITY

Cryptography is not magic... No need the secret key to decrypt your files If a keylogger or trojans run secretly on your computer !
PROTECT YOUR OS FIRST.
